# Shopify 

### Manual Installation

1. Log in to your Shopify admin panel

2. Install the following plugin in your shop, by clicking the install button on the following page:
https://apps.shopify.com/insert-code-to-header-and-footer

3. Click the little pin icon beside "XO Code Insert" so the app stays in the sidebar.

4.  Click on "Add new code":

<img src="/images/integrations/shopify/xo-add-code.png" />

5. Enter "mySHOEFITTER" into the "Title" input

6. Scroll down until you find the input box called "Code in body"

7. Place the code you received from us. It should look similar to this one:

```html filename="index.html" copy
<!-- Load the mySHOEFITTER Script -->
<script src="https://js.myshoefitter.com/v1/script.js"></script>

<!-- Initialize mySHOEFITTER -->
<script type="application/javascript">
  myshoefitter.init({
    shopSystem: 'shopify'
  });
</script>
```

<img src="/images/integrations/shopify/xo-code-editor.png" />

7. Click the save button and close the popup that appears after saving the code

8. Click on the link "theme customizer":

<img src="/images/integrations/shopify/xo-activate-1.png" />

9. Enable "XO Insert code Body" and click on "save":

<img src="/images/integrations/shopify/xo-activate-2.png" />

Now you should see the button on your product page. If not, please notify us, so a technican of our team can take a look.

### Installation Service

If you having trouble adding the script by yourself, we would love to help you.

1. In your Shopify admin area, go to Settings (Einstellungen)

2. Go to "Users and Roles" ("Nutzer und Berechtigungen")

3. Scroll down to Collaborators (Gäste)

4. Copy the for Digit Code and send it to us

<img src="/images/integrations/shopify/collaboration.png" />

<Callout type="info" emoji="ℹ️">
  Now, we can request access to your shop through our Shopify Partner Account. We won't be able to see any orders or customers. We just request permissions to install apps in your shop. You'll receive an email when we request access to your store. Please click the confirmation link in the email.
</Callout>
