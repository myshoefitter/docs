# HTML
 
Our JavaScript snippet is the easiest way to integrate mySHOEFITTER into your website.

### Add the Script

Add the following script tag add the end of your product pages, just before the `</body>` tag

```html filename="index.html" {3} copy
<script src="https://cdn.myshoefitter.com/script.js" defer></script>
```

### Add the Button

Add this code below your "add to cart" button:

```html filename="index.html" {3} copy
<button id="mySF_Widget_Button">Schuhgröße herausfinden</button>
```

### Expose the Product ID

Create a hidden html element that shows the product id in plain text.
Give it the id `product-id`.

```html filename="index.html" {3} copy
<div id="product-id" style="display: none;">ABCXYZ123</div>
```

That's it. You are ready!
