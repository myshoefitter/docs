# HTML
 
Our JavaScript snippet is the easiest way to integrate mySHOEFITTER into your website.

### Add the Script

1. Add the following script tag add the end of your product pages, just before the `</body>` tag

```html filename="index.html" {3} copy
<script src="https://cdn.myshoefitter.com/script.js" defer></script>
```

### Add the Button

Add a button below the "add to cart" button.

1. Add this code below your "add to cart" button:

```html filename="index.html" {3} copy
<button id="mySF_Widget_Button">Schuhgröße herausfinden</button>
```

That's it. You are ready!
